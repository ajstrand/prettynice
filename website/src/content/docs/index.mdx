## Features

<CardGrid>
	<Card title="Truly Full Stack" icon="seti:html">
        Static HTML rendered on the server.
        Client-side components for islands of interactivity in the browser.
        Both served in the same response.
	</Card>
	<Card title="Send, Don't Fetch" icon="rocket">
        Initialize components with data on the server that is automatically encoded and type-checked at compile time.
        No loading spinners. No runtime error checks.
	</Card>
	<Card title="Safety & Guarantees" icon="approve-check-circle">
        Strong static types and carefully managed side-effects thanks to the <a href="https://gren-lang.org/">gren programming language</a>.
	</Card>
	<Card title="Interop with Javascript" icon="seti:javascript">
        Easy access to the entire javascript ecosystem in the browser <em>and</em> in node!
	</Card>
</CardGrid>
